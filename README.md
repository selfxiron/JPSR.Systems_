# JPSR.Systems_

This is the source code for my personal portfolio. It’s a minimal, high-performance site built to showcase my transition into Machine Learning engineering and my focus on Mathematical Foundations.

## Under the Hood

I wanted to keep this project easy, so I decided to go with "no-framework" approach. It’s built using:

* **Vanilla JavaScript:** Handles all the core logic, theme switching, and interactive elements.
* **Tailwind CSS:** Used for the layout and responsive design.
* **HTML5 Canvas:** Powers the background animation.

### Key Features

* **Interactive Neural Network:** The background isn't a video or a GIF. It’s a live physics simulation of particles ("neurons") and connections.
* **Custom Cursor Physics:** Instead of the default pointer, the site uses a custom dual-node cursor that follows your movement with a slight mathematical lag (lerp) for a smoother feel.
* **Glassmorphism UI:** I used heavy backdrop blurs and semi-transparent layers to keep the content readable while letting the background simulation peek through.
* **Dark/Light Mode:** Includes a theme manager that remembers your preference using `localStorage`.
